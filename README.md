# itsm-knowledge-rag
Turns closed IT tickets into searchable, verifiable answers. RAG pipeline with PII redaction (Presidio + custom recognizers), hybrid retrieval (pgvector + BM25), and LLM curation of unstructured threads into unified problem statements — with a deterministic PII-leakage benchmark separate from judge-based curation quality (DeepEval/G-Eval).
